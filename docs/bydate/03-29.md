# 1941-03-29

Nubian）接近受损的意大利巡洋舰波拉号（Pola），俘虏了其船员，并用鱼雷将其击沉。

Gradisca 提供剩余幸存者的坐标，以继续救援。

英里处的盟军护航队 HX-115，击沉了 3 艘货轮。
