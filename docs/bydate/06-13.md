# 1940-06-13

# 1941-06-13

战斗机的空中掩护，英国的博福特鱼雷轰炸机队在战斗机护航下仍然能够突破并发起攻击，一枚鱼雷击中吕佐夫号的发动机室，对它造成了严重损坏。

海军陆战师的指挥权，成为隶属于美国海军大西洋舰队的第一军（临时）的指挥官。

威廉·哈里尔上校接替蒙哥马利上校担任美国海军"游骑兵"号驱逐舰的指挥官。
