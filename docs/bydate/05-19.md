# 1940-05-19

他的部队现在距离英吉利海峡仅 50 英里，并切断了比利时境内盟军的后路。

英国远征军司令戈特勋爵将军下令撤退到敦刻尔克等港口城市，而戴高乐上校的法国第四装甲师试图攻击古德里安在蒙科尔内的侧翼，但未能成功。

截至目前，英国皇家空军已损失了部署到法国和比利时的一半以上的飞机。

为了在可能入侵英国时保留实力，英国皇家空军开始召回国内剩余的飞机中队。

### 挪威
